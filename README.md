# GOBLIN

3D dungeon management sim built in Godot 4.4.x. Phase 1 prototype in progress.

You play a young goblin kid whose parents were murdered by adventurers. You wield a telekinetic ring that can pick up, direct, and slap your surviving minions. Mine, haul, build, defend — accumulate **Ruckus** (noise) — survive the raids it draws.
